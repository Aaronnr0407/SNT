Le Web et ses infrastructures

Dans le web, il y a aussi le Cloud, les Bases de données big data, SQL, cookies (aspect informationnel), RGPD...

Où sont stockées toutes les données du web ?

Les données du web comme les sites, les fichiers et les images sont tous stockés dans des data center aussi appelé base de données avec à l'intérieur des serveurs. C'est pour cela que nous pouvons voir des problèmes avec les sites quand le data center de Paris nommé Paris Digital Park qui possède plus de 7 hectares de serveurs. Ce que les gens apellent le cloud, c'est en réalité des serveurs qui stockent vos informations. Par exemple Onedrive est un cloud qui stocke vos informations et vos fichiers dans un serveur qui vous est dédié et qui permet de ne pas perdre vos informations si jamais vous les aviez stocké sur votre appareil. Sur la photo n°1, nous pouvons voir le Paris Digital Park vu de l'extérieur et sur la n°2, nous pouvons voir le data center de San José de l'intérieur. Vous vous demandez peut être quesqu'un serveur ?
Un serveur est le disque qui est connecté a internet. Donc un serveur est dans le data center et ce sont ces fameux serveurs qui vous traduisent du SQL vers le html que votre ordinateur lit comme dans le schéma ci dessous. On peut voir sur les 2 shémas le trajet d'une requête sur internet.

Qu'est ce que le SQL ?


Le SQL est le language infomatique que votre ordinateur envoie en requête sous forme de paquet au serveur. Une fois que le serveur a trouvé ce que vous recherchez, alors à ce moment là le serveur va vous renvoyer une requête en SQL qui sera automatiquement traduite par votre ordinateur. Vous acceptez tous le temps les cookies, mais savez-vous réellement ce que c'est ? Les cookies sont de petit fichiers texte enregistrés sur votre appareil, et qui va servir a votre appareil à vous faire des publicités plus ciblées. Comme si vous allez sur le site d'apple et que vous allez sur youtube alors youtube vous mettra plus facilement des pubs pour l'appareil apple. Un exemple dans les 2 documents ci dessous.

Le RGPD, qu'est-ce que c'est ?

Le RGPD c'est Le règlement général de protection des données (RGPD) c'est donc un texte réglementaire européen qui encadre le traitement des données de manière égalitaire sur tout le territoire de l'Union européenne (UE).
